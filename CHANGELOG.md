## [v0.1.0] - 2026-01-25

### Highlights

- Initial release of agent-team-prd: multi-agent system for PRD creation and review
- 16 specialized agents for discovery, specification, review, and revision workflows
- Integration with github.com/grokify/structured-requirements for PRD schema

### Added

- prdtool CLI for PRD management: init, validate, score, show, view commands
- prdtool-mcp MCP server exposing PRD operations as tools for AI assistants
- pkg/prd package wrapping structured-requirements types with helper functions
- pkg/scoring package for PRD quality scoring against rubric
- pkg/views package for PM and executive view generation
- 16 agent specifications: problem-discovery, user-research, market-intel, solution-ideation, requirements, ux-journey, tech-feasibility, metrics-success, risk-compliance, prd-lead, review-board, prd-scoring, exec-explainability, issue-extractor, revision-planner, change-validator
- Team workflow specification with DAG-based orchestration
- Deployment specifications for Claude Code, Kiro, Gemini, ADK, AWS Bedrock, CrewAI, and Kubernetes
- prdtool schema command for programmatic JSON Schema access from structured-requirements
- prd_schema MCP tool for AI agent schema access

### Dependencies

- github.com/grokify/structured-requirements v0.2.0 for PRD types and schema
- github.com/agentplexus/mcpkit v0.3.1 for MCP server runtime
- github.com/spf13/cobra v1.10.2 for CLI framework

### Documentation

- MkDocs documentation site configuration
- Agent specification markdown files with roles, responsibilities, and output formats
- README with project overview and quick start
